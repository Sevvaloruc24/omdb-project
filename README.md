# 🎬 MovieNexus - Modern Film Keşif Portalı

MovieNexus, OMDB API kullanarak geliştirilmiş, kullanıcıların binlerce film ve dizi arasında hızlıca arama yapmasını, detaylı bilgilere ulaşmasını sağlayan modern bir web uygulamasıdır.

## ✨ Özellikler

-   **Akıllı Arama:** Film veya dizi adlarını eksik girseniz bile (Örn: "Ed Gein") benzer sonuçları listeler.
-   **Gelişmiş Detay Paneli (Modal):** Bir filme tıkladığınızda; yönetmen, oyuncu kadrosu, IMDb puanı ve geniş konu özeti gibi bilgilere şık bir açılır pencere ile ulaşırsınız.
-   **Bunu mu demek istediniz? (Suggestive Search):** Aradığınız kelimeyle tam eşleşme bulunamadığında otomatik olarak alternatif öneriler sunar.
-   **LocalStorage Entegrasyonu:** Son yaptığınız aramayı hatırlar ve sayfayı yenilediğinizde size geri getirir.
-   **Responsive Tasarım:** "Glassmorphism" estetiği ile tasarlanmış arayüz, mobil cihazlarla tam uyumludur.

## 🛠️ Kullanılan Teknolojiler

-   **HTML5 & CSS3:** Modern grid ve flexbox yapıları, cam efekti (backdrop-filter) ve animasyonlar.
-   **Modern JavaScript (ES6+):** Fetch API, Async/Await mimarisi ve dinamik DOM yönetimi.
-   **OMDB API:** Gerçek zamanlı film veritabanı entegrasyonu.
